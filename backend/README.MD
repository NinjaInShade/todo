# Backend for the todo client.

## Todo API. Routes needed:

- Retrieve all todos (GET to '/api/todos/')
- Create new todo (POST to '/api/todos/')
- Update todo (PATCH to '/api/todos/{todoID}')
- Delete todo (DELETE to '/api/todos/{todoID}')
